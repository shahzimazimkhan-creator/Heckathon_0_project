# 🚀 5 AI Tools & Apps – Agentic Productivity Suite

An AI-powered multi-tool productivity dashboard built with **Streamlit + OpenAI API**.

This project contains 5 intelligent AI tools in one unified interface.

---

## 🧠 Features

### 1️⃣ Smart Email Assistant
- Summarize long emails
- Generate professional replies
- Improve tone and clarity

### 2️⃣ AI Data Insight Generator
- Upload CSV files
- Automatic statistics summary
- AI-generated insights
- Quick visualization chart

### 3️⃣ AI Contract Simplifier
- Upload PDF contracts
- Get simplified explanations
- Extract key obligations

### 4️⃣ AI Interview Coach
- Generate interview questions by role
- Evaluate your answers
- Get feedback and improvement tips

### 5️⃣ AI Personal Finance Planner
- Enter income & expenses
- Get AI budgeting advice
- Improve savings strategy
